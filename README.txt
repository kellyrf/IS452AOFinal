Welcome to the IS452AO Final Project, created by Kelly Ferreira for the Spring 2019 Semester.

This project focuses on creating a one day coding workshop in a public library. Instructional design, example workshop proposal, and
essential language that workshop participants will need for reference is provide in the Wiki of this GitHub. The file "Example Instructional Code.py" provided acts as a guide through the actual workshop. Sample promotional materials are included as well for the use 
of public libraries.
